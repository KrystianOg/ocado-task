# Approach


## Running locally:

Clone repo:

```bash
git clone https://github.com/KrystianOg/ocado-task.git
```

Install dependencies

```bash
npm install
```

Run development server:

```bash
npm run dev
```

Build for production

```bash
npm run build
```
